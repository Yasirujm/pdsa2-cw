# PDSA Game Suite (Full Stack Project)

## Features

- Snake and Ladder (Shortest Path Problem)
- Traffic Simulation (Maximum Flow Problem)
- Knight’s Tour (DFS and Greedy)
- Sixteen Queens Puzzle (Backtracking and Multithreading)
- Minimum Cost Assignment (Hungarian Algorithm and Min-Cost Max-Flow)

---

## Tech Stack

### Frontend
- Next.js
- Tailwind CSS

### Backend
- Spring Boot (Java 21)
- REST API

### Database
- MySQL

---

## Installation and Setup

### Frontend (Next.js)

Navigate to the frontend folder and run on terminal:

- npm install
- npm run dev

### Backend (Spring Boot)

Navigate to application.properties in resources folder and change DB credentials
